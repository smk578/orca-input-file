# orca-input-file
Work-in-progress fork with updates for compatibility with Orca 6+ input files

This is a Visual Studio Code (vscode) Extension which provides basic syntax highlightning for the ORCA 6 Package (Input files, *.inp*)
[ORCA 6 Manual](https://www.faccts.de/docs/orca/6.0/manual/index.html)

To install manually, copy the folder into C:\Users\\{XYZ}\\.vscode\extensions
